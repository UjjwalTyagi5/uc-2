## Instructions

Decide which quotation source the buyer most likely selected. Apply these criteria in order:

1. **Supplier match** — which supplier name is closest to the Primary Supplier or Parent Supplier above?
2. **Price match** — which quotation's extracted unit prices are closest to the RAS recorded prices?
3. **Latest date** — when supplier and prices are similar, prefer the most recent quotation date.

Return ONLY this JSON object — no markdown fences, no extra text:
