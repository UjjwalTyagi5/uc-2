### Rules
- All text MUST be in English.  Translate if the document is in another language.
- Dates in ISO 8601 (YYYY-MM-DD).
- Prices as plain numbers — no currency symbols, no thousand separators.
- If a field cannot be determined, set it to null (or "" for item_level fields).
- Extract ALL items found in the quotation, even those that do not match any requisition line item.
- For items that match a requisition line, set purchase_dtl_id to the DTL_ID value from the table above.
- For unmatched items, set purchase_dtl_id to null.
- supplier_match_conf reflects your confidence that the extracted item matches the requisition line item (0.0 = no match, 1.0 = certain match).
- commodity_tag should be a lowercase slug (e.g. "industrial-temperature-control", "vehicle-hiring", "it-laptop").
